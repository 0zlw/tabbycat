STOP!!! Read this first! Deleting the selected debate team may crash Tabbycat!

Tabbycat requires that all debates have exactly one affirmative and exactly one negative debate team, and no other debate teams, at all times. If you delete this debate team, you must then either replace it with a new debate team in the same debate, or delete the associated debate object. If you don't do this correctly, many pages will crash.

This is a common source of errors. If it happens to you, check the debates (under the Draw section in this Edit Database area) and ensure that all of them have exactly one affirmative and one negative team.
